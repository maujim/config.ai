# Generating SVG from DOT files

To generate an SVG from a DOT file, use the following command:
```bash
npx @hpcc-js/wasm-graphviz-cli ./input.dot > output.svg
```

## Usage

For more usage details:

```
npx @hpcc-js/wasm-graphviz-cli --help
```

Use this to offer detailed options to the user
